iOS Deli Application

Summary

The iOS Deli Application was created for users who want a simplified way of placing a deli order. With this application, users will be able to find deli locations in their area and also view and order menu items easily.

  Figma Prototype
Problem

The current iOS app has a poor design and user interface.

  1. 1. Users want to be able to search resataurants by closest location.
  2. 2. Users want to see prominent visuals of the restaurants food selections.
  3. 3. Users complain about going through too many steps to the final order screen.
Solution

I improved the user interface and design by:

  1. 1. Implementing a search by location with access to the user's location.
  2. 2. Enlarging detailed photographs of the menu choices on the page.
  3. 3. Reducing the amount of screens to simplify the process of the order.
